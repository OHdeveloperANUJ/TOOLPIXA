# ToolPixa Master Design System

## Typography
- Primary Font: "Inter" (for body, data, UI elements)
- Secondary Font: "Outfit" (for Display Headings, H1, H2)
- Monospace: "JetBrains Mono" (for code, math formulas)

## Color Palette (Premium Glassmorphic)
- Background: Very dark, premium slate/navy (`#0b1121`)
- Surface: Glassmorphic translucent layers with subtle white/blue borders (`rgba(255, 255, 255, 0.05)`)
- Primary Accent: Brilliant Electric Blue (`#3b82f6` to `#60a5fa`)
- Text Primary: Pure White (`#ffffff`)
- Text Secondary: Muted Slate (`#94a3b8`)

## Spacing System
Strict adherence to the 8px grid scale:
- Small elements: `8px`, `16px`
- Containers/Padding: `24px`, `32px`
- Sections: `48px`, `64px`, `96px`

## Animations & Transitions
- Micro-interactions on all interactive elements (buttons scale up 1.02x on hover).
- Page transitions: Fade in and slide up slightly.
- Data counting: When calculators produce results, the numbers should rapidly count up from 0.

## Borders & Radii
- Cards and Buttons should have generous corner radii (`12px` or `16px`).
- Borders should be ultra-thin (`1px`) and highly translucent (`border-white/10`).

## Icons
- Use Lucide React icons. Stroke width strictly `2px`.
